# Building Floor detection
The detection of floors using conventional methods is difficult.

# Requirements
python ==3.7.0
pytorch==1.6.0
opencv latest version
torchvision==0.7.1
matplotlib latest version
pillow 



# How to run
to train the model 

```
python train.py --batch_size 3
```

to eval and visualize results 

```
python eval.py 
```
